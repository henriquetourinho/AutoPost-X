AutoPost-X

Automatize seus posts com inteligência

Sobre o Projeto

O AutoPost-X é um bot automatizador para a rede social X (antigo Twitter) que evoluiu de um simples script em Shell para uma aplicação mais inteligente em Python.

Ele monitora perfis e feeds RSS, identifica conteúdos relevantes com base em filtros configuráveis, e publica automaticamente com diversas opções de interação.

Versão Shell Script (.sh)

Implementação inicial com funcionalidades básicas de automação, ideal para sistemas Unix-like.

Versão Python (.py)

Versão mais avançada com recursos adicionais, interface interativa e maior portabilidade entre sistemas.

Funcionalidades

Setup Interativo

Configuração fácil através de interface passo-a-passo no terminal.

Multiperfis e RSS

Monitora múltiplos perfis e feeds RSS simultaneamente.

Repost, Comentário, Like

Diversas formas de interação com os posts encontrados.

Conversão para Imagem

Transforma textos em imagens para melhor engajamento.

Filtros e Exclusões

Palavras-chave para incluir ou excluir conteúdos específicos.

Memória Anti-repetição

Registro de posts já processados para evitar duplicação.
